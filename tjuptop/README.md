# tjuptattendance

TJUPT 签到工具，支持 TOP10 签到

# 亮点

- 🥁 丰富的自定义参数
- ✨ 支持多用户
- 📧 邮件提醒，防止断签
- 🎈 断签检测及自动补签
- ⚡ 速度超快

# 如何使用

## 下载

1. 访问 [发布页面](https://github.com/azureqaq/tjuptattendance/releases) 下载最新的版本
2. 解压下载的压缩包，得到程序本体

## 运行

- `Windows`：打开 `cmd` 运行
- `Unix-like`：使用 `Terminal` 运行
